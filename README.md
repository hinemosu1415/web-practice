# web-practice

JavaScriptでHTMLを操作する練習用の課題集です。

各ディレクトリの `index.html` をブラウザで開き、`script.js` にコードを書いて動作を確認してください。
ブラウザのコンソールは `F12` で開けます。

## 課題一覧

### 1st-prac

JavaScriptの変数を使う練習です。

`script.js` に用意されている `text` の内容を `console.log` で出力してください。

### 2nd-prac

HTML要素のテキストを取得する練習です。

`index.html` の `<p>` タグに書かれている文章を、JavaScriptで取得して `console.log` で出力してください。

### 3rd-prac

入力欄とボタンを扱う練習です。

`Push` ボタンを押したときに、入力欄に書かれている内容を `console.log` で出力してください。

### 4th-prac

JavaScriptでHTML要素を追加する練習です。

`Push` ボタンを押したときに、`<ul>` タグの最後へ新しい `<li>` タグを追加してください。
追加する `<li>` タグの中身は `テキストを作成しました！` にしてください。

### 5th-prac

入力内容をリストに追加する練習です。

`追加` ボタンを押したときに、入力欄に書かれている内容を `<li>` タグとして `<ul id="list">` に追加してください。

## 進め方

1. 各課題の `index.html` をブラウザで開く
2. `script.js` にコードを書く
3. ブラウザを再読み込みして動作を確認する
4. 必要に応じてコンソールでエラーを確認する
